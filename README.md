### 🕹️ Game Features

#### Core Gameplay
- **Mine Trilium (TLM):** Use your NFT tools to mine for the game's native currency, TLM, on various planets.
- **Stake to Planets:** Stake your TLM to one of six Planet DAOs to increase the planet's reward pool and gain voting power.
- **Participate in Governance:** Vote in planetary elections to determine strategies and control of planetary treasuries.
- **NFT-Based Missions:** Send virtual spaceships (using NFT cards) on missions to compete for a share of a TLM reward pool.

#### What Makes Alien Worlds Special
- ⭐ **Deep Economic Simulation:** A complex interplay between mining, staking, and governance creates a true DeFi experience.
- 🎨 **Cross-Chain Functionality:** Operates across WAX, Ethereum, and BNB Smart Chain, allowing for a vast and interconnected economy.
- 🤝 **Decentralized Autonomous Organizations (DAOs):** The world is run by six Planet DAOs, putting governance directly in the hands of players.
- 🔗 **Interoperable NFTs:** All in-game items (Tools, Avatars, Weapons, Land) are NFTs that can be freely traded.

### 💡 Troubleshooting

- **Installation fails:** Run as Administrator/sudo and temporarily disable any antivirus software.
- **Simulator won't launch:** Ensure you have the latest version of your operating system and web browser.
- **Calculations are slow:** Close other background applications, especially other browser tabs.
- **NFTs not appearing:** Use the "Refresh NFT Database" button within the tool's settings menu.
